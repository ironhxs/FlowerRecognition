## 3. 训练策略

### 3.1 数据预处理

#### 图像预处理
```python
- 调整大小：600x600
- 归一化：使用ImageNet统计量 (mean=[0.485, 0.456, 0.406], std=[0.229, 0.224, 0.225])
- 格式转换：RGB, Tensor
```

#### 数据增强
训练时使用的数据增强技术：
1. **几何变换**
   - RandomResizedCrop (scale=0.7-1.0)
   - RandomHorizontalFlip (p=0.5)
   - RandomVerticalFlip (p=0.3)
   - ShiftScaleRotate (rotate_limit=45度)

2. **颜色增强**
   - ColorJitter (brightness=0.3, contrast=0.3, saturation=0.3)
   - RandomBrightnessContrast

3. **高级增强**
   - GaussianBlur
   - GaussNoise
   - CoarseDropout (Cutout)
