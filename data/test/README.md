# Test Data | 测试数据

Place your test images here, organized by class.

请将测试图像放在这里，按类别组织。

## Structure | 结构

```
test/
├── class1/
│   ├── image1.jpg
│   ├── image2.jpg
│   └── ...
├── class2/
│   ├── image1.jpg
│   ├── image2.jpg
│   └── ...
└── ...
```

Each subdirectory represents one flower class.

每个子目录代表一个花卉类别。
